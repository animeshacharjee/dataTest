Objective

In this problem two data sets are given. One of the data set is acting as training data (pml-training) and other one is test data (pml-testing).  In the training data,  "classe" acting as response variable which really made up of different types of classes such as “A”, “B”, “C” etc. We need to build a model with the training data sand predict the “Class” (ID Numbers) in the test data

Data quality checks
